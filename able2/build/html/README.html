What Is A/B Testing?¶

A/B testing is a method of comparing two versions of a webpage or app against each other to determine which one performs better.

Vocabulary¶

Assuming there is no underlying difference between A and B, how often will we see a difference like we do in the data just by chance? The answer to that question is called the significance level, and “statistically significant results” mean that the significance level is low, e.g. 5% or 1%

N = sample size
 X = success
 $H_{0}$ =

  • Null hypothesis

  • $p{exp} - p{cont} = 0$

$H_{A}$ =

  • alternative hypothesis

  • $p{exp} - p{cont} <> 0$

The goal is to have a $P(p_{exp}-p{cont} | H_0)$ small enough we reject null if p<0.05 ($p<alpha$)

$alpha$ =

  • probability of falsely concluding there was a difference

  • P(reject null | null true)

  • conventionally set at .05

  • false positives

  • Type I errors

$beta$ =

  • power

  • propability of failing to reject the null when the null was in fact false

  • P(fail to reject | null false)

$1-beta$ =

  • sensitivity or

  • power of the experiment

  • you often want 80%

d_{min} =

Small sample:

null true

null false

reject null

alpha low

fail to reject

beta high

  • alpha is low : unlikely to launch a bad experiment

  • beta is high : likely to launch an experiement that actually have a difference you care about

Large sample:¶

null true

null false

reject null

alpha same

fail to reject

beta lower

  • distribution get tighter

  • alpha doesn’t change

  • unlikley to fall in the range of failing to reject the null

  • more likely to reject the null and conclude there was a difference

What is statistical power?

The power of any test of statistical significance is defined as the probability that it will reject a false null hypothesis. Statistical power is inversely related to beta or the probability of making a Type II error. In short, power = 1 – β.

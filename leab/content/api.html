    Module leSample¶

    class leab.before.leSample.leSample(conversion_rate: float, min_detectable_effect: float, significance_level: float = 0.05, statistical_power: float = 0.8, absolute: bool = True)[source]¶

    Bases: object

    Build leSample object.

    Parameters

    • conversion_rate (float) – baseline conversion rate.

    • min_detectable_effect (float) – minimum detectable effect.

    • significance_level (float) – alpha, percent of the time a difference will be detected, assuming one does NOT exist.

    • statistical_power (float) – 1-beta, percent of the time the minimum effect size will be detected, assuming it exists.

    Example

    >>> from leab import before

>>> ab_test = before.leSample(conversion_rate=20,
...                           min_detectable_effect=2)
>>> ab_test.get_size()
6347

>>> ab_test.get_duration(avg_daily_total_visitor=1000)
13

    Module leSuccess¶

    class leab.after.leSuccess.leSuccess(sample_A: pandas.core.frame.DataFrame, sample_B: pandas.core.frame.DataFrame, confidence_level: float = 0.95)[source]¶

    Bases: leab.after.leSuccess.Chi2Sample

    Build leSuccess object.

    Parameters

    • sample_A (pd.DataFrame) – A sample data.

    • sample_B (pd.DataFrame) – B sample data.

    • confidence_level (float) – desired confidence level, default : 95%.

    Example

    >>> from leab import leDataset
>>> from leab import after

>>> data = leDataset.SampleLeSuccess()
>>> ab_test = after.leSuccess(data.A,
...                           data.B,
...                           confidence_level=0.95)
>>> ab_test.sample_A.confidence_interval

[8.526343659939133, 22.13718821096384]

>>> ab_test.p_value

0.25870176105718934

>>> ab_test.get_verdict()

'No significant difference'
